# Proyecto JAVA EV 3

Este proyecto es un Sistema de Punto de Venta (POS) desarrollado en Java con el entorno NetBeans, diseñado específicamente para la administración de inventario y registro de ventas de cartas coleccionables (Trading Card Games - TCG).

Permite a los usuarios (vendedores y administradores) iniciar sesión, gestionar el stock de productos, registrar nuevas ventas y generar reportes sobre el desempeño del negocio.

Funcionalidades Clave:

✅ Login Seguro: Acceso diferenciado para vendedores y administradores.  
✅ Gestión de Stock: Visualización, búsqueda y actualización de inventario (ID, Nombre, Tipo, Precio y Stock).

✅ Registro de Ventas: Interfaz amigable para seleccionar productos del stock, agregarlos al carrito y generar el total de la venta.  
✅ Administración de Usuarios: Funcionalidad para añadir, modificar y listar nuevos vendedores o administradores (usando el RUT como identificador).

✅ Reportes: Módulo de reportes para visualizar el total de ventas e ingresos generados.
